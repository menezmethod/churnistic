# API Documentation

## Overview

Churnistic's API is built using tRPC, providing type-safe endpoints for automated bank bonus and credit card tracking with minimal user input.

## Core Principles

1. Minimal User Input
   - Auto-populated data
   - Smart defaults
   - One-click actions
   - Automated tracking

2. Fast Development
   - Simple endpoints
   - Type safety
   - Clear responses
   - Quick setup

## Authentication

Firebase authentication required for all endpoints:

```typescript
const client = createTRPCClient({
  headers: {
    Authorization: `Bearer ${firebaseIdToken}`,
  },
});
```

## MVP Endpoints

### Bonus Tracking

#### 1. Quick Add

```typescript
// Add bonus from URL
mutation.bonus.quickAdd({
  url: string; // DoC or Reddit URL
});

// Add bonus manually
mutation.bonus.create({
  bankId: string;
  amount: number;
  requirements: {
    directDeposit?: {
      amount: number;
      frequency: 'one-time' | 'monthly';
    };
    balance?: {
      amount: number;
      duration: number;
    };
    transactions?: {
      count: number;
      type: string;
    };
  };
});

// Get active bonuses
query.bonus.getActive({
  status?: 'planned' | 'in_progress' | 'completed';
});
```

#### 2. Progress Tracking

```typescript
// Update progress automatically
mutation.bonus.updateProgress({
  bonusId: string;
  transactions: Transaction[];
});

// Get bonus details
query.bonus.getDetails({
  bonusId: string;
});

// Get upcoming deadlines
query.bonus.getDeadlines({
  daysAhead?: number;
});
```

### Automation

#### 1. Browser Extension

```typescript
// Capture bonus from webpage
mutation.capture.fromPage({
  url: string;
  content: string;
});

// Validate captured data
query.capture.validate({
  capturedData: CapturedBonus;
});
```

#### 2. Notifications

```typescript
// Set up notifications
mutation.notifications.setup({
  email?: string;
  browser?: boolean;
  preferences: {
    newBonuses: boolean;
    deadlines: boolean;
    completions: boolean;
  };
});

// Get notification status
query.notifications.getStatus();
```

## Data Types

### Core Types

```typescript
interface Bonus {
  id: string;
  userId: string;
  bankId: string;
  amount: number;
  status: 'planned' | 'in_progress' | 'completed';
  startDate: Date;
  targetDate: Date;
  requirements: {
    directDeposit?: {
      amount: number;
      frequency: 'one-time' | 'monthly';
      completed: boolean;
      progress: number;
    };
    balance?: {
      amount: number;
      duration: number;
      completed: boolean;
      current: number;
    };
    transactions?: {
      count: number;
      type: string;
      completed: boolean;
      progress: number;
    };
  };
  autoTracking: {
    enabled: boolean;
    emailForwards?: string[];
    statementSync?: boolean;
  };
}

interface Transaction {
  date: Date;
  amount: number;
  description: string;
  type?: string;
}

interface CapturedBonus {
  url: string;
  bank: string;
  amount: number;
  requirements: {
    type: string;
    details: string;
  }[];
  expirationDate?: Date;
  restrictions?: string[];
}
```

## Error Handling

```typescript
// API Error Types
type APIError =
  | 'INVALID_INPUT'      // Invalid request data
  | 'NOT_FOUND'          // Resource not found
  | 'UNAUTHORIZED'       // Not authenticated
  | 'AUTOMATION_FAILED'  // Automation error
  | 'SCRAPING_FAILED';   // Scraping error

// Error Response
interface ErrorResponse {
  code: APIError;
  message: string;
  details?: any;
}
```
